# Gazebo Testing Platform

## How to use
1. Install the following packages:  
    - gazebo_ros
    - turtlebot3_navigation
    - turtlebot3_gazebo
    - gmapping
    - rviz  

2. Then using gazebo to create a map and scan it with ```robot_mapping``` package
3. Run the navigation in ```robot_navigation```
